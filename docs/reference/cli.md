# CLI reference

## `gestalt`

Runs the dashboard/API server.

Key options and environment variables are documented by `gestalt --help` and in the sections below.

## `gestalt-agent`

Runs a standalone agent from `.gestalt/config/agents/*.toml`.

```sh
gestalt-agent <agent-id>
gestalt-agent <agent-id> --dryrun
```
